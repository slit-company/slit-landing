# SPEC Web

비개발자 운영팀도 직접 관리할 수 있도록 설계한 실운영형 커뮤니티 플랫폼

## 개요

SPEC Web은 외부 브랜딩 사이트와 내부 운영 도구를 하나의 제품으로 통합한 Next.js 기반 플랫폼이다. 단순 소개 페이지가 아니라 관리자 대시보드, 출석 관리, 과제 운영, 콘텐츠 발행, 기록 축적 기능을 함께 다루는 운영 시스템으로 설계했다. 리포지토리 기준 개발 기간은 2026년 2월 8일부터 2026년 4월 13일까지이며, 총 478 commits가 축적되었다. 실제 운영에 사용된 플랫폼으로, 웹사이트 제작을 넘어 운영 체계 자체를 제품 안에 내장한 사례다.

## 클라이언트 과제

클라이언트가 필요로 한 것은 보기 좋은 웹사이트 하나가 아니었다. 외부 방문자에게는 신뢰도 있는 브랜드 경험이 필요했고, 내부 운영팀에게는 콘텐츠 수정, 출석 체크, 과제 관리, 활동 기록 누적을 한곳에서 처리할 수 있는 관리 도구가 필요했다. 특히 운영 주체가 바뀌더라도 개발자 의존 없이 페이지와 데이터를 관리할 수 있어야 했고, 흩어진 도구를 오가며 생기는 운영 비용도 줄여야 했다. 단기 홍보 사이트가 아니라 장기 운영 가능한 플랫폼을 만드는 일이 핵심 과제였다.

## 솔루션

이 프로젝트에서는 Next.js 15 App Router 기반 구조 위에 퍼블릭 페이지와 관리자 기능을 함께 배치하고, Supabase를 통해 인증과 데이터 흐름을 일원화했다. 비개발자 관리자 대시보드를 별도 기능으로 두어 운영팀이 직접 콘텐츠를 수정하고 게시할 수 있게 했으며, BlockNote와 Tiptap을 결합해 문서형 콘텐츠 작성 경험도 강화했다. 운영 측면에서는 출석 관리와 과제 관리를 같은 대시보드 안으로 통합했고, Padlet 연동으로 외부 협업 흐름을 연결했다. 또 스레드형 로그 시스템을 설계해 활동 기록이 단발성 공지로 사라지지 않고 지속적으로 축적되도록 만들었다.

## 기술 스택

| 영역 | 사용 기술 |
| --- | --- |
| Frontend | Next.js 15, TypeScript, Tailwind CSS |
| Backend / Runtime | Next.js App Router, Supabase, `middleware.ts` |
| Editor / Authoring | BlockNote, Tiptap |
| Data / Integration | Supabase, Padlet 연동 |
| Testing / Infra | Playwright, Vercel |

## 아키텍처

리포지토리는 `app/`, `components/`, `lib/`, `supabase/`, `middleware.ts`를 축으로 구성된다. `app/`은 퍼블릭 페이지와 로그인 이후 운영 화면을 포함하는 라우팅 레이어로, 외부 노출 영역과 내부 관리 영역을 단일 코드베이스 안에서 함께 다룬다. `components/`는 대시보드 UI, 편집기 인터페이스, 공통 화면 요소를 모듈 단위로 분리해 재사용성을 높이고, `lib/`는 데이터 접근, 유틸리티, 도메인 로직을 정리하는 계층으로 작동한다. `supabase/`는 인증과 데이터 저장, 운영 기능에 필요한 백엔드 연동을 담당하며, `middleware.ts`는 접근 제어와 세션 기반 분기를 처리해 관리자 기능과 일반 사용자 흐름을 구분한다. 전체적으로는 콘텐츠 관리, 운영 자동화, 권한 흐름을 App Router 구조 안에서 일관되게 관리할 수 있도록 설계된 아키텍처다.

## 핵심 구현 하이라이트

- 비개발자도 직접 페이지와 운영 데이터를 다룰 수 있는 관리자 대시보드를 구축해 개발 의존도를 낮춤
- 출석 관리와 과제 운영을 하나의 플랫폼으로 통합해 반복 운영 업무를 웹 안에서 일관되게 처리
- BlockNote와 Tiptap 기반 편집 환경을 붙여 문서 작성과 게시 경험을 제품 내부로 끌어옴
- Padlet 연동으로 외부 협업 도구와 내부 운영 흐름이 끊기지 않도록 연결
- 스레드형 로그 시스템을 설계해 활동 기록과 과제 이력을 구조적으로 축적할 수 있게 구현
- Playwright 기반 검증 체계를 두고 Vercel에 배포해 실제 운영 환경 반영 속도를 높임
